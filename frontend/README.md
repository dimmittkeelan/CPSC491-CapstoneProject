# 🖥️ PC Build Generator

A web-based application that helps users generate optimized PC builds based on their budget and preferences. The system simplifies the custom PC building process by recommending compatible components and providing a clean interface.

---

## 🚀 Overview

Building a custom PC can be overwhelming due to compatibility issues, pricing, and performance considerations. PC Builder automates the process by:

- Generating builds based on user's budget
- Validating compatibility between components
- Allowing users to save and manage builds
- Providing a modern and responsive user interface

---

## ✨ Features

- 🔐 User Authentication (Login / Signup)
- 💰 Budget-Based Build Generation
- 🧩 Component Compatibility Validation
- 📦 Component Breakdown View
- 💾 Saved Builds Dashboard
- ⚡ Responsive UI
- 🌐 Deployment via Vercel
